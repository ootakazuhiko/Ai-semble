# Ai-semble

AIによるセミ構造化タスク自動化プロジェクト

## 概要
本プロジェクトは、AIを活用してWeb業務やデータ処理などのセミ構造化タスクを自動化するための基盤を提供します。

## セキュリティ機能
- 暗号化モジュール
- 認証・認可（RBAC）
- 監査ログ
- セキュリティ設定管理

## テスト・CI
- pytestによる自動テスト
- GitHub Actions + Codecovによるカバレッジ計測
